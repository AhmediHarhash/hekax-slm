# HEKAX SLM Engine

## Domain-Specific Small Language Models

The HEKAX SLM Engine develops domain-specific small language models optimized for fast, low-cost inference across voice and text communication workflows.

---

## Internal Applications

- Real-time automation
- Vertical reasoning
- Call intent classification
- Message drafting
- State handling
- Summarization

---

## Objectives

- Enable deterministic, domain-focused reasoning
- Reduce latency to sub-150ms
- Support on-device or single-GPU deployment
- Improve cost/performance for large-scale usage
- Support multi-model routing strategies

---

## Components

- Fine-tuning pipeline
- Evaluation suite
- Quantization scripts
- Dataset builders
- Industry task datasets
- Benchmark reports
